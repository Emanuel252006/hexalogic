# HexaLogic - Sitio Web

Sitio web para HexaLogic creado con HTML y CSS puro.

## Características

- ✅ Diseño responsivo
- ✅ Menú de navegación con indicador de página activa
- ✅ Footer con información de contacto y redes sociales
- ✅ Botón flotante de WhatsApp
- ✅ Logo animado con efecto de rodado
- ✅ 4 páginas principales: Inicio, Servicios, Portafolio y Contacto
- ✅ CSS moderno con gradientes y efectos

## Servicios

- **Desarrollo Web:** Front-End (React, HTML, CSS), Back-End (Node.js), Bases de datos (SQL Server, MySQL y PostgreSQL)
- **Desarrollo de Sistemas de Escritorio:** Implementación con C#
- **Desarrollo de Aplicaciones Móviles:** Samari Front, Java y Flutter
- **Asesoramiento Técnico:** Front-End, Back-End y bases de datos
- **Soporte Técnico:** 8 horas diarias, 5 días a la semana

## Archivos

### Páginas HTML
- `index.html` - Página de inicio
- `servicios.html` - Página de servicios
- `portafolio.html` - Página de portafolio
- `contacto.html` - Página de contacto

### Archivos CSS
- `css/base.css` - Estilos base comunes (header, footer, botones, animaciones del logo, etc.)
- `css/index.css` - Estilos específicos para la página de inicio
- `css/servicios.css` - Estilos específicos para la página de servicios
- `css/portafolio.css` - Estilos específicos para la página de portafolio
- `css/contacto.css` - Estilos específicos para la página de contacto

## Cómo usar

1. Abre `index.html` en tu navegador web
2. Navega entre las páginas usando el menú
3. Usa el botón de WhatsApp para contacto directo
4. Completa el formulario de contacto

## Personalización

- Cambia el logo editando los SVG en las páginas HTML
- Modifica los colores editando las variables CSS
- Actualiza la información de contacto en el footer
- Añade tus propios proyectos en `portafolio.html`
